# BAAAHS website

Dependencies::::::
1) node: (https://nodejs.org/en/download/)[https://nodejs.org/en/download/]

To run site:
`git clone [URL OF THIS REPO]`
`cd baaahs.org`
`npm install`
`npm run build`
`npm start`

Navigate to (http://localhost:1337)[http://localhost:1337] to view this site locally.
